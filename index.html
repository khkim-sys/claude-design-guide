01
방법 1 — 기본 과정
디자인 시스템 없이 바로 만들기
빠르게 제작이 필요할 때 · 가장 기본적인 방법 · 약 5분
STEP BY STEP
1
기획 내용 정리(Claude Chat 또는 Cowork 활용하여 스토리보드 만들기)
Claude Chat 또는 Cowork를 활용해 보다 완성도 높은 스토리보드와 보고자료를 생성할 수 있도록 자료에 포함될 내용을 상세히 작성하여 요청해 보세요!
⚠️ 주의: 클로드 챗·코워크에서 스토리보드를 요청할 때 반드시 '텍스트로만 작성해 달라'고 명시하세요. 이미지 디자인을 직접 출력하려 할 수 있습니다.
💡 핵심: 가장 중요한 것은 탄탄한 스토리보드와 설계입니다. 구조가 명확하고 내용이 잘 정리될수록 Claude Design이 더 완성도 높은 결과물을 만들어냅니다.
보고자료에 들어갈 내용을 넣은 다음 (텍스트 또는 참고할 이미지, 엑셀 내용 등) 아래 프롬프트를 입력하세요.
💬 "첨부한 데이터를 기반으로 분석 결과 및 인사이트를 전달하는 PPT 슬라이드 보고자료를 제작합니다. 슬라이드에 사용할 스토리 보드 초안을 작성하세요. 각 슬라이드에는 키 메시지와 근거 데이터가 명확히 작성되어야 합니다. 또한 각 슬라이드에는 분석 결과를 효과적으로 시각화하는 시각화 자료도 포함입니다."
2
레퍼런스 수집(선택)
Figma 커뮤니티, 핀터레스트 등에서 참고할 디자인 이미지 2~3장을 미리 다운받아 준비합니다.
3
Claude Design 접속
claude.ai 접속 → 상단 또는 사이드바에서 Design 클릭하여 진입합니다.
4
Slide deck 탭 → New slide deck 생성
좌측 탭에서 Slide deck 선택 후 팝업에서 아래 항목을 설정합니다.
●Project name — 프로젝트명 입력
●Design system — 디자인 시스템을 구축하지 않은 경우, 기존 선택 항목의 × 버튼을 눌러 해제하거나 그대로 두면 됩니다. (Org default · no assets yet 상태면 영향 없음. 'None' 옵션은 없음)
●Use speaker notes — 발표자 노트가 필요하면 토글 활성화
●Create 버튼 클릭
5
프롬프트 입력 + 레퍼런스 첨부
클로드 챗·코워크에서 작성한 기획 내용을 붙여넣고, 준비한 레퍼런스 이미지도 함께 첨부합니다.
💬 "이번 5월 전사 구성원 대상 월간미팅 PPT 자료(총 8장) 초안을 작성해주세요. 첨부한 레퍼런스 디자인과 아래 내용을 참고해서 전체적인 디자인 톤앤매너와 슬라이드별 텍스트 배치를 기획해주세요."
6
수정 및 완성
초안 생성 후 Claude Design 내 Edit 기능에서 텍스트·도형 배치를 직접 수정하거나, PPT 파일로 다운로드하여 수정합니다.
TIP 애니메이션이 들어간 슬라이드 만들기
1Claude Design 접속 → 우측 Skills 패널에서 Make a deck 추가
2채팅창에 아래 프롬프트 입력
💬 "첨부한 자료와 위 스토리보드를 기반으로 발표용 슬라이드를 생성합니다. 방향키로 서브 스텝이 순차적으로 등장하는 인터랙티브한 발표 자료로 생성하세요. 추가로 확인이 필요한 항목이 있다면 저와 함께 꼼꼼히 검토한 후 제작을 시작합니다."
3방향키로 서브 스텝이 순차적으로 등장하는 인터랙티브한 애니메이션 슬라이드가 생성됩니다
02
방법 2 — 심화 과정
디자인 시스템 반영하여 만들기
브랜드 일관성이 필요할 때 · 약 5분 구축 시간 필요
PHASE 1 — Figma에서 fig 파일 준비
1
기획 내용 정리(Claude Chat 또는 Cowork 활용하여 스토리보드 만들기)
Claude Chat 또는 Cowork를 활용해 보다 완성도 높은 스토리보드와 보고자료를 생성할 수 있도록 자료에 포함될 내용을 상세히 작성하여 요청해 보세요!
⚠️ 주의: 클로드 챗·코워크에서 스토리보드를 요청할 때 반드시 '텍스트로만 작성해 달라'고 명시하세요. 이미지 디자인을 직접 출력하려 할 수 있습니다.
💡 핵심: 가장 중요한 것은 탄탄한 스토리보드와 설계입니다. 구조가 명확하고 내용이 잘 정리될수록 Claude Design이 더 완성도 높은 결과물을 만들어냅니다.
보고자료에 들어갈 내용을 넣은 다음 (텍스트 또는 참고할 이미지, 엑셀 내용 등) 아래 프롬프트를 입력하세요.
💬 "첨부한 데이터를 기반으로 분석 결과 및 인사이트를 전달하는 PPT 슬라이드 보고자료를 제작합니다. 슬라이드에 사용할 스토리 보드 초안을 작성하세요. 각 슬라이드에는 키 메시지와 근거 데이터가 명확히 작성되어야 합니다. 또한 각 슬라이드에는 분석 결과를 효과적으로 시각화하는 시각화 자료도 포함입니다."
2
Figma에서 템플릿 수집 → fig 파일 변환
Figma 로그인 → Community → 디자인 템플릿 선택 → Use template → 팝업 하단 '모든 슬라이드 추가' → File > Save local copy 로 deck 파일 저장.
⚠️ 주의: Claude Design 디자인 시스템에는 .deck 파일이 아닌 .fig 파일만 업로드 가능합니다.
①템플릿 전체 선택 후 복사 (Ctrl+A → Ctrl+C)
②Figma 홈 → 상단 Design 클릭 → 새 파일 열기
③빈 파일에 붙여넣기 (Ctrl+V)
④File > Save local copy → .fig 파일로 저장됨
PHASE 2 — Claude Design 시스템 구축
3
디자인 시스템 생성
claude.ai/design 접속 → 상단 탭 Design systems → Create 버튼 클릭.
4
시스템 구성 내용 작성
Create 화면을 캡처해서 클로드 챗에게 .fig 파일과 함께 아래와 같이 요청합니다.
💬 "저는 Claude Design 시스템을 제작하려고합니다. 첨부한 .fig 파일을 분석하고, 캡처된 화면에 들어갈 내용(브랜드 정보, 컬러, 폰트, 컴포넌트 등)을 작성해주세요."
챗에서 나온 내용을 바탕으로 디자인 시스템 입력창에 작성 후 등록합니다.
5
시스템 자동 구축 완료 확인약 5분 소요
등록 후 채팅창이 열리며 생성이 시작됩니다. 완료 후 디자인 관련 체크 항목들이 표시됩니다.
PHASE 3 — 슬라이드 제작
6
디자인 시스템 기반으로 자료 제작
💬 "슬라이드에 들어갈 내용과 함께 구축된 디자인 시스템에 맞게 첨부된 기획안 내용을 보고 자료로 만들어 주세요."
Claude Design이 디자인 시스템을 읽고 자동으로 PPT를 생성합니다.
💡 Claude Design 사용량 절약 팁
1
먼저 5장만 테스트 — 전체 슬라이드를 한 번에 요청하지 말고, "일단 5장만 테스트로 작업해 주세요"로 시작하세요. 방향을 확인한 뒤 나머지를 요청하면 불필요한 재작업을 줄일 수 있습니다.
2
간단한 수정은 Edit에서 직접 — 텍스트 내용 수정, 색상 변경 등 간단한 작업은 채팅으로 요청하지 말고 Claude Design 내 Edit 기능에서 직접 수정하세요. 채팅 요청 없이 수정할 수 있어 사용량을 아낄 수 있습니다.
3
스토리보드 완성 후 Design 진입 — Claude Chat에서 스토리보드를 먼저 완성한 뒤 Design으로 넘어오세요. Design 내에서 긴 대화를 이어가는 것보다 사용량을 크게 절약할 수 있습니다.
4
재생성 대신 프롬프트 수정 — 결과가 마음에 들지 않으면 새 메시지를 추가하지 말고 기존 프롬프트를 직접 수정해서 재생성하세요. 대화 길이가 늘어나지 않아 토큰이 절약됩니다.
7
수정 및 완성
Claude Design 내 Edit에서 직접 수정하거나, PPT 파일로 다운로드하여 수정합니다.
